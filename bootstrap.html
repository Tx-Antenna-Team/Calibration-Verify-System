⚠️ หน้านี้ใช้สำหรับตั้งค่าครั้งแรกเท่านั้น
หลังจากสร้าง Admin เสร็จแล้ว ให้ลบไฟล์ bootstrap.html ออกจาก repository ทันที
ขั้นตอน:
1. กรอกรหัสพนักงาน Admin คนแรก (เช่น MTL00001)

3. กรอกรหัสผ่านอย่างน้อย 6 ตัวอักษร
4. กด "สร้าง Admin Account"
→ ไปยังหน้า Login
